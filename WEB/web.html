Project Web

Assignment: A online interactive travel guide

Design a online interactive travel guide. This travel guide is based on a customer journey map and data from a real person. Take you customer journey as basis, and design a guide for multiple devices. Design and build a prototype with a traveling API where all your data that you put in the prototype comes from.

Project: Web

Date: September 4, 2019

Target group: Traveling students

Customer Journey map

Conclusions from the customer journey map:

  • Before travelling, the user looks up things to do in his/her phone.
  • The user is always looking for reviews, opening and closing times and the cost for the activities.
  • The user wants to be able to save an activity, so the user is able to look it back up without searching.

Prototype

Early filtering

Filtering unnecessary information, by making choices on a early stage so the user doesn't have to scan through things they don't want to.

Looking up reviews

Easy reviews navigation for quick and helpful reviews

My list

Always able to look back at the list you saved. Able to look up information online and offline at any given moment!

Looking up information

Straight to the point interface, so the user can find all the information the user wants to see from a travel guide!

Changing device

Done with searching on the computer? No problem, save your list you made on the computer, and get it all back with a code on your phone!
